# 🐛 Ticketing systém: Anketa (Operační systémy)

Vítejte! Tento repozitář neslouží pro zdrojové kódy, ale funguje výhradně jako **ticketingový systém pro hlášení chyb a návrhů na vylepšení** mého školního projektu (předmět Webové aplikace).

## 🌐 Kde najdete aplikaci k testování?
Moje anketa běží živě zde: **[https://hoch.pythonanywhere.com/](https://hoch.pythonanywhere.com/)**

## 🛠️ Jak mi nahlásit chybu nebo vylepšení?
Pokud najdete jakoukoliv chybu, nebo vás napadne, co by se dalo vizuálně či funkčně zlepšit, postupujte takto:

1. Nahoře v menu tohoto repozitáře klikněte na záložku **[Issues](../../issues)**.
2. Klikněte na zelené tlačítko **New issue**.
3. Do názvu stručně popište problém (např. *„Chybí potvrzení o hlasování“* nebo *„Překlep na hlavní stránce“*).
4. Do popisu detailněji rozepište, co je špatně a jak by to mělo fungovat.
5. Odešlete kliknutím na **Submit new issue**.

Jakmile ticket vytvoříte, já se na něj podívám, pokusím se ho opravit na serveru a následně ticket slavnostně uzavřu. 

Díky moc za pomoc s testováním! 🚀
